# プロマーケター視点によるサイトレビュー｜平沼柊哉 様

全セクション（Hero / About / Services / Plans / Strengths / Projects / Career / Contact）を確認しました。総合的に**「清潔感があり読みやすい、しかし刺さりきっていない」**サイトという印象です。以下、項目ごとに詳しくレビューします。

---

## 1. キャッチコピー ★★★☆☆

**現状：**「ITのこと、まるごとお任せください。」

**良い点：**
ターゲット（IT初心者の経営者・中小企業）に向けた平易な言葉で、方向性は正しい。

**課題：**

- 「まるごとお任せ」は他のIT系フリーランスサイトでもよく使われる汎用表現で、**差別化になっていない**
- ベネフィット（あなたにとって何が得られるか）が不明確。「任せたら何が変わるのか」が伝わらない
- 副キャッチ「御社の『IT担当』として、なんでもお手伝いします」は良い方向性だが、こちらのほうがむしろ**メインに使える独自性**がある

**改善案の方向性：**
「社内にIT担当者がいない。そのまま損してませんか？」「月額固定で、ITをまるごと外注できる時代へ。」など、**ペインポイントを突いたコピー**にするとCVRが上がりやすい。

---

## 2. 構成 ★★★★☆

**現状の流れ：**
Hero → About（自己紹介）→ Services → Plans → Strengths → Projects → Career → CTA → Contact

**良い点：**

- 概ね王道のLP構成になっており、読み進めやすい
- 「自社サービス」セクションで実績・信頼性を補完している点は◎
- 最終CTAの前にキャリア経歴を置き、信頼感を積み上げる流れは理にかなっている

**課題：**

- **「About（自己紹介）」がHeroの直後**にくるのは早すぎる。訪問者は「自分の悩みが解決できるか」を最初に知りたいのであり、あなたの経歴は後でいい
- **「選ばれる理由（Strengths）」がPlansの後**にある。本来は「なぜあなたに頼むべきか」を先に納得させてから価格を見せるべき
- **お客様の声・実績・受注実績数がゼロ**。これが最大の構成上の欠落。信頼構築の核になる

**推奨構成：**
Hero → 課題提起（ターゲットのペイン） → Services → Strengths（選ばれる理由）→ 実績・お客様の声 → About（経歴で後押し）→ Plans → CTA

---

## 3. デザイン ★★★★☆

**良い点：**

- ダークネイビー×ホワイトの配色は清潔感・プロフェッショナル感があり、ITサービスのトーンに合っている
- 余白の使い方が上手く、読みやすい
- ナビゲーションがシンプルで迷わない
- 料金プランのカード型UIは比較しやすく◎
- モバイル対応も問題なさそうなレイアウト

**課題：**

- **ヒーローセクションに視覚的なインパクトが弱い**。写真・動画・イラストなど視覚的要素が一切なく、テキストのみ。ファーストビューで人の顔が見えると信頼感・親近感が大幅にアップする
- Aboutセクションのプロフィール写真が小さく、白バックのシンプルすぎる写真。もう少し印象的・親しみやすい写真にすると好感度が上がる
- **アイコンがすべて同系統の細線アイコン**で、サービスの違いが視覚的に区別しにくい
- 「自社サービス」の各サービス画像のビジュアルトーンがバラバラ（Tabliaはきれいだが、酒学道場・こよいのワインは雰囲気が異なる）

---

## 4. 問い合わせまでの導線 ★★☆☆☆

**現状：**

- ナビバーに「お問い合わせ」ボタン（常時表示）
- Hero直下に「まずは相談してみる」CTA
- 各料金プランに「無料で相談する」ボタン
- ページ最下部に「まずはお気軽にご相談ください」セクション

**良い点：**

- CTAが複数箇所にある点は良い

**課題（最重要課題）：**

- **お問い合わせフォームが「お名前・メールアドレス・内容」の3項目のみ**。これは最低限すぎて、むしろ「何を書けばいいかわからない」という摩擦を生みやすい
- **フォームにプレースホルダー（入力例）がない**。「お問い合わせ内容」に何も例示がなく、初めての訪問者が入力をためらう
- **送信後の動線が不明**。「送信したらどうなるの？いつ返信来るの？」という不安を消す一文がない（例：「24時間以内にご返信します」）
- **Calendlyなどの日程調整ツール連携がない**。「いきなりメール」はハードルが高い。「30分無料相談をその場で予約できる」導線があると一気にCVRが上がる
- 「無料で相談する」というCTAコピーは良いが、**全プランで同じテキスト**のため差がない

---

## 5. ビジネスモデル ★★★★☆

**現状：**

- 月額顧問型（ライト15万・スタンダード25万・プレミアム40万）＋スポット開発（Webサイト30万〜、Webアプリ50万〜、既存改修20万〜）

**良い点：**

- **ストック収益（月額顧問）をメインに置いている**のは非常に賢い。フリーランスにとって安定収益基盤となる
- 「1ヶ月から始められる」という縛りのなさは初めての顧客に刺さる
- スポット開発を別立てにすることで、顧問に踏み切れない顧客も拾える

**課題：**

- **15万〜40万という価格帯の妥当性が説明されていない**。なぜこの価格なのかの根拠（工数・対応時間など）が不透明で、初見では「高いかも」と思われるリスクがある
- **スポット開発の価格もWebサイト30万〜と幅が広すぎる**。「Webサイト（5ページ）30万円」などの具体例があると比較・判断しやすい
- **自社サービス（Tablia等）とコンサルティング事業の関係性が整理されていない**。訪問者から見ると「この人は開発も受けるの？SaaSを作ってる人なの？」と混乱しやすい。「自社プロダクトも持っているからこそ、事業視点でサポートできる」という文脈を明示するべき

---

## 6. 見せ方（訴求力・ポジショニング） ★★★☆☆

**良い点：**

- 「外注なし、全工程一人で完結」という差別化ポイントは明確
- 「専門用語を使わず、わかりやすくご説明します」は中小企業オーナーへの刺さるメッセージ
- 積水ハウスという大企業での経験を記載しており、信頼性を補強している

**課題：**

- **数字・実績がほぼゼロ**。「支援実績〇社」「開発プロジェクト〇件」「顧問歴〇年」「満足度〇%」などの数字が一切ない。これが最も信頼獲得のボトルネックになっている
- **ターゲットが絞り切れていない**。中小企業なのか、スタートアップなのか、個人事業主なのか。ターゲットが広い分、誰にも刺さりきらないリスクがある
- **競合との差が「一人でやる」だけになっている**。これは強みにも弱みにも読める（「一人だと万が一の時が不安」という懸念を植え付けるリスクもある）。そこへの反論（「だからAIと仕組みで補完している」など）をセットで見せるべき
- **SNS・ブログ等の外部コンテンツへの導線がない**。個人ブランディングで最も効いてくる「日々の発信」との接続がなく、サイト単体で完結しているため、再訪・ファン化のループが生まれにくい

---

## 総合評価と優先改善アクション

| 優先度    | 改善項目                                           |
| --------- | -------------------------------------------------- |
| 🔴 最優先 | 実績数字・お客様の声を追加する                     |
| 🔴 最優先 | お問い合わせフォームにCalendly等の日程予約を追加   |
| 🟠 高     | キャッチコピーをペインドリブンに変更               |
| 🟠 高     | 構成を「Strengths → 実績 → Plans」の順に変更       |
| 🟡 中     | ヒーローセクションにビジュアル（写真・動画）を追加 |
| 🟡 中     | フォームにプレースホルダー・返信目安を追加         |
| 🟢 低     | ターゲット絞り込みまたはターゲット別ページ設計     |

**全体として：デザインとコンテンツの質は及第点以上。ただし「信頼の証拠」と「問い合わせへの摩擦除去」に集中して改善するだけで、CV率は大きく変わるはずです。**
